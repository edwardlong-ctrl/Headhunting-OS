# Contracts Package

Shared contracts for the Recruiting Transaction OS.

Future work should define versioned schemas and API/event contracts here before implementing product behavior in services or UI.

Task 2B adds the first contracts-first truth-layer skeleton under `schemas/`. These files define shared enum and object contract shapes only; they do not implement service behavior, UI behavior, demo data, AI model connections, matching, consent/disclosure behavior, or client-safe DTO generation.
